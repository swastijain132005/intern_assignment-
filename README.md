# 🌿 Vibo ERP – SaaS Landing Page

A modern, responsive landing page for **Vibo ERP**, a cloud-based ERP solution designed for fruit & vegetable wholesalers and traders.

## ✨ Features

- 📊 End-to-End Voucher Lifecycle (Invoices, Payments, Receipts)
- 📱 One-Click WhatsApp Automation
- 📈 Financial Reports (Balance Sheet, P&L)
- 📦 Inventory & Business Reporting
- 🔒 Role-Based Access Control (UI representation)
- ⚖️ Weight-Based Invoicing (for traders)

---

## 🎯 Key Highlights

- Clean and modern SaaS UI design
- Fully responsive layout (mobile-friendly)
- Consistent color system and typography
- Smooth animations using Framer Motion
